# Arbitrary Points Allocator

## Description

Manually sets company points each in-game day based on configured values, enabling custom league table scoring.

---

## Features

- Custom league table for all companies
- Daily manual point assignment via settings
- Automatic handling of company creation, renames, mergers, and bankruptcies
- Save and load safe

---

## How It Works

Each in-game day, the script reads the configured point values and directly sets each company’s score in the league table.

---

## What's It For?

Custom challenges, leagues, tournaments, and any scenario where scoring should be fully manual rather than calculated by the game.

---

## Installation

1. Place the `ArbitraryPointsAllocator` folder into your `OpenTTD\content_download\game` directory
2. In OpenTTD, go to **Game Script Settings** and select _Arbitrary Points Allocator_
3. During the game, click the **League Table button** to view company standings
